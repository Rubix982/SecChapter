# like1000

## Description

This [.tar file](https://jupiter.challenges.picoctf.org/static/52084b5ad360b25f9af83933114324e0/1000.tar) got tarred a lot.

## Solution

```sh
curl -LO https://jupiter.challenges.picoctf.org/static/52084b5ad360b25f9af83933114324e0/1000.tar
```

You can use the `extractor.sh` script to extract all the way down.

Eventually, you get `flag.png`.

Flag -> `picoCTF{l0t5_0f_TAR5}`.
